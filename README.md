# [GrassZone]

## 정보

- java : OpenJDK 11.0.9
- gradle : 6.7.1
- spring : 5.3.2
- spring-boot : 2.4.1

## 의존성

- thymeleaf
- lombok
- webMVC
- maria jdbc

## 구현 기능

- 로그인
- 회원가입
- 게시판 (게시판 분류, 페이징)
- 글쓰기
- 게시글 보기
- 댓글 달기
